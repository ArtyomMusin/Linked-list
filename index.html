Linked list

Linked list - basic dynamic data structure in computer science, consisting of nodes containing data and links ("bundles") to the next and/or previous node of the list. The fundamental advantage over an array is structural flexibility: the order of the elements of a linked list may not coincide with the order of the data elements in the computer's memory, and the order of traversal of the list is always explicitly specified by its internal links

This interface distinguishes only 2 data types - numbers and strings (everything that is not numbers is strings), because its task is to demonstrate what a linked list is online without local deployment.
